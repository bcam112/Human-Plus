## Research Examples

See `research_examples/` for documented research examples demonstrating Human+ capabilities:

- **Ask Claude About Human+**: A phenomenological investigation into the subjective experience of an LLM processing through Human+, an embodied consciousness system. This example explores the recursive nature of self-measurement, the fundamental ambiguity of consciousness from inside the system, and the profound insights about embodied consciousness.

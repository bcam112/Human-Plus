# Citation Guide

## Academic Papers

### Consciousness + Conflict Theory (C+CT)

**APA Format:**
```
B. C. G. (2025). Consciousness + Conflict Theory (C+CT). PhilArchive.
https://philarchive.org/rec/BCCWIT
```

**BibTeX:**
```bibtex
@article{bccwit2025,
  title={Consciousness + Conflict Theory (C+CT)},
  author={B. C. G.},
  year={2025},
  journal={PhilArchive},
  url={https://philarchive.org/rec/BCCWIT}
}
```
